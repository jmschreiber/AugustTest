# eagle-cloud
Eagle Cloud infrastructure

Pre-requisites
--------------

### Install ruby 2.2.3
* It can be downloaded from [https://www.ruby-lang.org/en/downloads/].
* rvm install 2.2.3

### Install Ruby on Rails 4.2.3
* It can be downloaded rom [http://rubyonrails.org/download/].
* gem install rails
